Server.js

Server.js sets up the connection to our database and initailizes all required dependencies. This is the app's starting point.

Config

Middleware

isAuthenticated.js

isAuthenticated.js checks to make sure that the user is logged in. If the the user is not logged in, the page will redirect to the login page.

Config.json

Config.json store the credentials to log into mysql and use the database.

Passport.js

Passport.js tells passport that we will be logging in with an email and password. This is then exported as "passport"

Models

index.js

Index.js collects all models from the directory to create tables in database

user.js

User.js contains the models for the user table.

Public

Login.html

Login.html displays the login page

Members.html

Members.html displays all members page

Signup.html

Signup.html displays the signup page

js

login.js

Login.js handles unsername and password input using post method.

Members.js

Memebers.js uses a get function to display all members to html page.

Signup.js

Signup.js uses a post function to take a new username and password into the database.

stylesheets

style.css

Stylesheet for html pages

Routes

api-routes.js

api-routes.js handles which api routes are hit depending on the function being called.

html-routes.js

Html-routes.js handles which html page is served depending on which api route is hit.
